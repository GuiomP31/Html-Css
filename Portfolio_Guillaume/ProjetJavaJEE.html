Projet Java JEE

Le but de ce projet était de faire un blog sur des recettes de cuisine.

L'utilisateur avait pour choix de s'inscire, de commenter et de voter au sujet d'une recette. Même non inscrit un utilisateur pourra donner son avis pour recette.

Système d’exploitation : Linux

Langage : Java EE 7

Utilisation de Pencil pour le visuel, Utilisation de Gantt Project, Trello pour l’organisation du projet

IDE : NetBeans 8.2, Serveur : Apache-Tomcat

Gestionnaire version : GIT avec dépôt GitHub, Communication : Discord

Présentation orale : powerpoint
